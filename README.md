# Home_Art
Online store for home appliances.

This project was created using Mern stack, for the backend we used Node.js With express and mongodb as Database, for the frontend we used React with the famous component librerie Material-ui 4.0 the project gonna be uptaded to the new version of MUI 5.0 with some minor changes to make it more robust.
